# 有益效果

本发明相比现有单向量列索引技术具有四方面显著的有益效果。第一，双向量列HNSW索引通过单一索引结构同时管理两列向量的图结构数据，避免了为每列单独创建索引后再进行外部结果融合的重复IO开销。传统方法需要对两列分别执行近邻搜索，再进行集合操作和分数重算，而本发明将两列检索结果在索引扫描过程中实时融合，显著降低了磁盘读取次数和中间结果集大小。第二，多ORDER BY词典序排序的按需计算机制避免了次列距离的全量计算。在传统实现中，即使主列排序结果已经确定，仍需对全部候选行计算次列距离用于去 tie，而本发明仅在主列距离相等时才触发次列计算，大幅减少了距离函数的调用次数。第三，RRF算法的流式批处理与理论上限剪枝相结合，一方面通过固定批次大小控制在任意时刻的内存占用，避免一次性加载全部候选集，另一方面通过数学上界严格判定提前终止条件，在保障结果正确性的前提下减少不必要的索引扫描轮次，配合Leader-Worker并行架构进一步缩短查询响应时间。第四，查询优化器集成框架对用户完全透明，用户无需学习新的SQL语法或手动指定执行计划Hint，优化器在规划阶段自动识别RRF查询并重写为CustomScan路径，降低了使用门槛，使得具备索引感知能力的融合排序能够无缝接入现有应用系统。
